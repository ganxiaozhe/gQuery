gQuery

DOM 操作

简介

DOM（Document Object Model——文档对象模型）是用来呈现以及与任意 HTML 或 XML 文档交互的 API。DOM 是载入到浏览器中的文档模型，以节点树的形式来表现文档，每个节点代表文档的构成部分。（例如：页面元素、字符串或注释等等）

gQuery 原型链中皆在为 DOM 服务，换言之：若不在 DOM 环境下，gQuery 原型链将无法正常运行，但这并不影响其拓展链。

索取与给予

尽管这小节的名字有点弦外之音的意思，但 gQuery 确实为 DOM 信息捕获与增减变化提供了更多可能。

例如用于获取内容与变化的函数方法：

方法名说明示例
text([:String:Array])获取或设置元素文本内容$('p').text()
html([:String:Array])获取或设置元素 HTML 内容$('p').html('g<b>Query</b>')
ohtml([:String:Array])获取或设置元素 outerHTML 内容$('p').ohtml([])
val([:String:Array])获取或设置元素值（表单）$('input').val([])
width([:Number:String])获取或设置元素宽度$('p').width('50vh')
height([:Number:String])获取或设置元素高度$('p').height()
css(:Object:String [, :String])获取或设置元素样式$('p').css({color:'red', opacity:.7})
offset([:Object:Boolean])获取元素定位坐标或设置元素 style$('#footer').offset()
hasClass(Class:String)判断元素（任一）是否有指定类$('p').hasClass('p1')
addClass(Class:String)为元素添加指定类$('p').addClass('p1 p2 p3')
removeClass(Class:String)为元素移除指定类$('p').removeClass('p2')
toggleClass(Class:String)为元素添加或移除指定类$('p').toggleCass('p1 p3')

以及用于元素增减的方法：

方法名说明示例
remove([Selector:String, onlyChildren:Boolean])移除元素$('h2').remove()
empty([Selector:String])清空元素$('h2').empty()
prepend(Element:String:Object)将内容插入到元素前（内）$('h2').prepend('<p>Hey!</p>')
append(Element:String:Object)将内容插入到元素后（内）$('h2').append('<p>Hi!</p>')
before(Element:String:Object)将内容插入到元素前（外）$('h2').before('<p>Sup!</p>')
after(Element:String:Object)将内容插入到元素后（外）$('h2').after('<p>Hello!</p>')

用于信息捕获及设置的方法：

方法名说明示例
attr(:Object:String [, :String])获取或设置元素属性值$('p').attr('data-name', 'val')
removeAttr(:String)移除元素指定属性$('p').removeAttr('data-name')
data(:Object:String [, :String])获取或设置元素的储存数据$('p').data({name:'甘小蔗', age:20})
removeData(:String)移除元素指定的储存数据$('p').removeData('name age')

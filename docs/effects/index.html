gQuery 特效

介绍

基于 gQuery 内置的动画计时器 $.fn.animate，提供了一些常用的动画特效。

显示 / 隐藏

$.fn.show([:String]) 与 $.fn.hide() 可快捷显示或隐藏元素。其中 show 方法支持传入一个字符以强制设置其 display 属性。

渐变

$.fn.fadeIn() 与 $.fn.fadeOut() 可使元素逐渐淡入或淡出，同时提供了 $.fn.fadeToggle() 来自动切换这两种状态。以上三种方法均接受两个可选参数来控制动画持续时间及回调：(duration:Number, callback:Function)。

滑动

$.fn.slideDown() 与 $.fn.slideUp() 可使元素逐渐滑动以展开或收起，同时提供了 $.fn.slideToggle() 来自动切换这两种状态。以上三种方法均接受两个可选参数来控制动画持续时间及回调：(duration:Number, callback:Function)。

动画计时器

以上内置的动画特效皆基于动画计时器 $.fn.animate(props:Object [, options:Number:Object, callback:Function]) 以实现。

$('h2').animate({opacity:0});

$('h2').css('position', 'relative').animate({
    opacity:0.5, left:'100px'
}, 1500, function(){
    console.log('Done.');
});

需要注意的是，该方法目前仅支持简单的数值类特效，不支持复杂数值如 RGB 动画等，如有相关需求请使用 Velocity。gQuery 并不打算支持 JS 动画的实现，而是推荐通过改变元素类以 CSS 来实现动画。

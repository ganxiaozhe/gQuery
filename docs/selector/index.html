gQuery 选择器

介绍

gQuery 选择器支持 CSS3 选择器 对 DOM（Document Object Model——文档对象模型） 单个元素或元素组进行操作。简而言之，gQuery 选择器其实就是：

document.querySelectorAll(selector) => 
gQuery.fn.init(selector) => 
gQuery(selector) => 
$(selector)

通过 $(selector).length 可以获取捕获元素的数量，通过 $(selector)[index] 获取相应 Element 。

尝试：在控制台执行 $('p') 以及 $('p').length 查看返回内容

注意：gQuery 选择器运行时需要 DOM Element 已经存在，请控制代码在 Document ready 之后被执行。gQuery 提供 $(function(){ //需要运行的代码 }); 以在 DOMContentLoaded 之后执行该匿名函数。

find 筛选

$.fn.find(Selector:String [, selfContain:Boolean, onlyChildren:Boolean]) 接受一个 CSS3 选择器文本 以筛选得到当前匹配的元素集合中每个元素下符合条件的后代。

console.log( $('p').find('code').text([]) );
/* selfContain */
$('p').find('p').length;    /* 0 */
$('p').find('p', true).length;  /* 10 */

eq 筛选

$.fn.eq(:Number) 接受一个 数值 以筛选得到当前匹配的元素集合中指定顺序的元素。

console.log( $('p').eq(2).text() );

关系筛选

gQuery 提供 $.fn.parent() 和 $.fn.next([Selector:String]) 选择元素集合中每个元素的 父级 和 下一兄弟级 元素。

在使用 $.fn.next() 时，若某个元素没有下一兄弟级元素，将其不会返回任何元素节点。

let $elems = $('p');
console.log(` original:${$elems.length} \n next:${$elems.next().length}`);

exist 判断

$.fn.exist() 可用于判断当前匹配的元素集合（任一元素）是否存在于 DOM。

let $elem = $('p').eq(0);
$elem.length>0 && console.log('元素存在');
$elem.exist() && console.log('[exist] 元素存在');

/* 从 DOM 中移除元素 */
$elem.remove();

$elem.length>0 && console.log('元素存在');
$elem.exist() && console.log('[exist] 元素存在');

is 判断

$.fn.is(Selector:String) 接受一个 CSS3 选择器文本 以判断当前匹配的元素集合中是否有符合条件的元素。

console.log( $('p').eq(0).is(':first-of-type') );    /* true */
console.log( $('p').eq(1).is(':first-of-type') );   /* false */
